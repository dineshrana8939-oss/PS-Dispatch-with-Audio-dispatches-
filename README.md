# PS-Dispatch-with-Audio-dispatches-
Custom Audio Dispatch for PS-Dispatch. Adds .ogg audio alerts to existing dispatch notifications. Credits to PS Sloth for original PS-Dispatch. Add audio to InteractSound and ps-dispatch sounds, include in fxmanifest, set name in shared/config.lua. Use admin test commands after adding: add_ace group.admin command.ps-dispatch-test allow.
